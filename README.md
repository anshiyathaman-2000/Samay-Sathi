Samay Saathi — AI Deadline Companion

Har deadline se pehle, aapka AI saathi.

An intelligent productivity companion that helps students, professionals and entrepreneurs never miss a deadline again — through smart AI prioritization, auto-scheduling, and context-aware reminders.

Features AI-powered task prioritization (urgency + importance + effort) Auto-scheduler — fits tasks into your working hours automatically Context-aware reminders at 24h / 6h / 1h thresholds Voice input support (Hindi + English) Habit & streak tracker Analytics dashboard Bilingual — English & Hindi 100% offline — data stays in your browser

Quick Start pip install flask python server.py

Tech Stack Python · Flask · Vanilla JavaScript · CSS3 · Web Speech API

No database. No login. No data leaves your device.
